## Description

This folder contains the GPT configuration files from before the PDF generation template improvements were introduced. These files represent the stable baseline that included:

- Full quotation capabilities with BOM and accessories
- Sales evaluation and training features
- Shopify catalog integration
- BROMYROS pricing optimization
- Accessories catalog (70+ items)
- BOM rules (parametric, 6 construction systems)
- Autoportancia table (unified)

## What This Version Does NOT Include

- ❌ PDF template v2.0 mandatory rules (logo, comments block, banking info)
- ❌ Detailed `unit_base` calculation logic in GPT instructions
- ❌ `build_quote_pdf()` canonical function reference
- ❌ Standardized technical nomenclature enforcement (`Thickness_mm`, `Length_m`)
- ❌ Enhanced PDF quality checklist
