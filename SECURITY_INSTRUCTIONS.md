# Guía de Seguridad Paso a Paso

## ¿Es esto seguro?
**SÍ.** Estamos creando una "bóveda" dentro de tu computadora para guardar las contraseñas.
NUNCA guardamos las contraseñas en archivos de texto planos que los hackers puedan leer.

## Instrucciones para configurar

He preparado un comando para ti en la terminal. Cuando lo apruebes:

1.  Aparecerá un menú:
    ```
    1. OPENAI_API_KEY
    2. WOLF_API_KEY
    ...
    ```
2.  Escribe `1,2` y presiona **Enter**.
3.  El programa te pedirá la `OPENAI_API_KEY`.
    *   **Pégala** y presiona **Enter**.
    *   *NOTA:* Por seguridad, no verás nada mientras escribes. Es normal.
4.  El programa te pedirá la `WOLF_API_KEY`.
    *   **Inventa una contraseña segura** (ej: `Lobo-Alfa-2025!`) y escríbela.
    *   Presiona **Enter**.

Una vez veas el mensaje "Proceso finalizado", avísame para lanzar el sistema.
